# Sales API

API RESTful para gerenciamento de vendas, construída com .NET 8.0 seguindo princípios de Clean Architecture, DDD e boas práticas de código.

## Tecnologias

- .NET 8.0
- C#
- PostgreSQL
- MongoDB (mockado)
- EF Core
- MediatR
- AutoMapper
- xUnit + NSubstitute
- Swagger

## Estrutura do Projeto
